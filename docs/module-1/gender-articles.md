# Le genre et les articles

En espagnol, chaque nom est masculin ou féminin. Il n'existe pas de neutre.
Les articles et les adjectifs s'accordent toujours avec le nom.

## Les articles définis

| | Singulier | Pluriel |
|-|-----------|---------|
| Masculin | el | los |
| Féminin | la | las |

*el libro* — le livre · *los libros* — les livres
*la casa* — la maison · *las casas* — les maisons

## Les articles indéfinis

| | Singulier | Pluriel |
|-|-----------|---------|
| Masculin | un | unos |
| Féminin | una | unas |

*un perro* — un chien · *unas flores* — des fleurs

## Reconnaître le genre

**Généralement masculins :**
- terminaison en **-o** : libro, vaso, perro
- terminaison en **-or** : color, calor
- jours, mois, langues, nombres

**Généralement féminins :**
- terminaison en **-a** : casa, mesa, silla
- terminaison en **-ción / -sión** : nación, televisión
- terminaison en **-dad / -tad** : ciudad, libertad

::: warning Exceptions courantes
Certains mots en **-a** sont masculins : *el problema*, *el mapa*, *el día*, *el idioma*.
:::

## Le pluriel

| Terminaison | Règle | Exemple |
|-------------|-------|---------|
| Voyelle | + s | casa → casas |
| Consonne | + es | ciudad → ciudades |
| -z | -z → -ces | vez → veces |

## Contraction obligatoire

**a + el = al** · **de + el = del**

*Voy al mercado.* — Je vais au marché.
*El coche del vecino.* — La voiture du voisin.

::: tip
La contraction ne s'applique pas avec **él** (pronom) : *Hablo de él.* — Je parle de lui.
:::
