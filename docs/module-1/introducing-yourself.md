# Se présenter

## Vocabulaire essentiel

| Espagnol | Français |
|----------|----------|
| me llamo... | je m'appelle... |
| mi nombre es... | mon nom est... |
| tengo ... años | j'ai ... ans |
| soy de... | je suis de... |
| vivo en... | je vis à... |
| soy + nationalité/profession | je suis + nationalité/profession |
| hablo... | je parle... |
| encantado / encantada | enchanté(e) |
| mucho gusto | avec plaisir / enchanté |

## Structures de base

**Se nommer**
> *Me llamo María.* — Je m'appelle María.
> *Mi nombre es Carlos.* — Mon nom est Carlos.

**L'âge**
> *Tengo treinta años.* — J'ai trente ans.

::: tip
En espagnol on « a » un âge avec **tener** (avoir), pas « être » comme en français.
:::

**L'origine**
> *Soy de Francia.* — Je suis de France.
> *Vivo en Madrid.* — Je vis à Madrid.

**La nationalité** (accorde avec le genre)

| Masculin | Féminin | Français |
|----------|---------|----------|
| francés | francesa | français(e) |
| español | española | espagnol(e) |
| inglés | inglesa | anglais(e) |
| alemán | alemana | allemand(e) |

> *Soy francés.* / *Soy francesa.*

**Les langues**
> *Hablo francés y un poco de español.* — Je parle français et un peu d'espagnol.

## Dialogue type

> — *¡Hola! Me llamo Ana. ¿Y tú?*
> — *Me llamo Pierre. Soy de Francia, vivo en Lille.*
> — *¡Encantada! ¿Hablas español?*
> — *Un poco. Estoy aprendiendo.* — Un peu. J'apprends.

## Pratique

Écris ta propre présentation en espagnol en utilisant :
- `me llamo`
- `tengo ... años`
- `soy de` / `vivo en`
- `hablo`
