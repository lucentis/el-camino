# Demander et comprendre

## Demander son chemin

| Espagnol | Français |
|----------|----------|
| ¿Dónde está...? | Où est... ? |
| ¿Cómo se va a...? | Comment on va à... ? |
| ¿Hay un/una... cerca? | Il y a un/une... près d'ici ? |
| a la derecha | à droite |
| a la izquierda | à gauche |
| todo recto | tout droit |
| la primera calle | la première rue |
| el semáforo | le feu de signalisation |
| la esquina | le coin de rue |
| cerca / lejos | près / loin |
| a pie | à pied |
| en metro / en autobús | en métro / en bus |

**Exemples :**
> *¿Dónde está la estación de metro?* — Où est la station de métro ?
> *¿Hay un supermercado cerca?* — Il y a un supermarché près d'ici ?
> *Gire a la derecha en el semáforo.* — Tournez à droite au feu.

## Au restaurant

| Espagnol | Français |
|----------|----------|
| Una mesa para dos, por favor. | Une table pour deux, s'il vous plaît. |
| ¿Qué recomienda? | Qu'est-ce que vous recommandez ? |
| ¿Qué lleva este plato? | Ce plat contient quoi ? |
| Sin..., por favor. | Sans..., s'il vous plaît. |
| Soy alérgico/a a... | Je suis allergique à... |
| ¡Buen provecho! | Bon appétit ! |
| La cuenta, por favor. | L'addition, s'il vous plaît. |
| ¿Está incluido el servicio? | Le service est inclus ? |

## Ne pas comprendre

| Espagnol | Français |
|----------|----------|
| No entiendo. | Je ne comprends pas. |
| ¿Puede repetir, por favor? | Vous pouvez répéter, s'il vous plaît ? |
| Más despacio, por favor. | Plus lentement, s'il vous plaît. |
| ¿Cómo se dice... en español? | Comment dit-on... en espagnol ? |
| ¿Qué significa...? | Que signifie... ? |
| ¿Habla francés / inglés? | Vous parlez français / anglais ? |
| Estoy aprendiendo español. | J'apprends l'espagnol. |
