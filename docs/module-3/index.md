# Module 3 — Le quotidien

**Prérequis :** [Module 2 — Construire une phrase](../module-2/)

**Objectif :** enrichir son vocabulaire et se débrouiller dans les situations du quotidien — faire ses courses, demander son chemin, décrire les choses.

Les bases grammaticales sont posées. Ce module élargit le vocabulaire et introduit les verbes irréguliers indispensables.
