# Verbes irréguliers courants

Les verbes irréguliers ne suivent pas les terminaisons standard. Ils sont incontournables — il faut les mémoriser.

## ir (aller)

| Pronom | Forme |
|--------|-------|
| yo | voy |
| tú | vas |
| él / ella / usted | va |
| nosotros | vamos |
| vosotros | vais |
| ellos / ustedes | van |

> *Voy al mercado.* — Je vais au marché.
> *¿Adónde vas?* — Tu vas où ?

## tener (avoir / tenir)

| Pronom | Forme |
|--------|-------|
| yo | tengo |
| tú | tienes |
| él / ella / usted | tiene |
| nosotros | tenemos |
| vosotros | tenéis |
| ellos / ustedes | tienen |

> *Tengo hambre.* — J'ai faim.
> *¿Tienes hermanos?* — Tu as des frères et sœurs ?

::: tip
**saber** vs **conocer** — deux verbes pour "savoir/connaître" :
- **saber** = savoir faire quelque chose, connaître un fait
- **conocer** = connaître une personne, un lieu

*Sé hablar español.* — Je sais parler espagnol.
*Conozco Madrid.* — Je connais Madrid.
:::
