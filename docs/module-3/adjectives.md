# Les adjectifs

## L'accord

En espagnol, les adjectifs s'accordent en genre et en nombre avec le nom qu'ils qualifient.

| | Masculin | Féminin |
|-|----------|---------|
| Singulier | un chico alto | una chica alta |
| Pluriel | unos chicos altos | unas chicas altas |

**Adjectifs en -e ou consonne** — invariables en genre, seulement le nombre change :

| | Singulier | Pluriel |
|-|-----------|---------|
| | un hombre inteligente | unos hombres inteligentes |
| | una mujer inteligente | unas mujeres inteligentes |

## La place de l'adjectif

**Après le nom** — règle générale :

> *una casa grande* — une grande maison
> *un coche rojo* — une voiture rouge

**Avant le nom** — pour exprimer une qualité inhérente ou stylistique :

> *la blanca nieve* — la blanche neige
> *un buen amigo* — un bon ami

### Adjectifs qui changent de sens selon la place

| Adjectif | Avant le nom | Après le nom |
|----------|-------------|--------------|
| grande → gran | un gran hombre — un grand homme (remarquable) | un hombre grande — un homme grand (physiquement) |
| mismo | el mismo día — le même jour | el día mismo — le jour en question |
| viejo | un viejo amigo — un vieil ami (de longue date) | un amigo viejo — un ami âgé |
| nuevo | un nuevo coche — une nouvelle voiture (différente) | un coche nuevo — une voiture neuve |
| pobre | un pobre hombre — un pauvre homme (malheureux) | un hombre pobre — un homme pauvre (sans argent) |

## Formes abrégées

Certains adjectifs perdent leur dernière syllabe devant un nom masculin singulier :

| Forme complète | Forme abrégée | Exemple |
|----------------|---------------|---------|
| bueno | buen | un buen día |
| malo | mal | un mal momento |
| grande | gran | un gran problema |
| primero | primer | el primer piso |
| tercero | tercer | el tercer módulo |
