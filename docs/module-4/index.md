# Module 4 — Parler du temps

**Prérequis :** [Module 3 — Le quotidien](../module-3/)

**Objectif :** parler du passé, du futur, et situer des événements dans le temps.

L'espagnol distingue deux passés là où le français n'en a souvent qu'un. Ce module explique quand et pourquoi choisir l'un plutôt que l'autre.
