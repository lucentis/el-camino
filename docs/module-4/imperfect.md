# Le pretérito imperfecto

L'imperfecto décrit des états, des habitudes ou des actions en cours dans le passé.
Il s'oppose à l'indefinido qui exprime des actions ponctuelles et terminées.

## Conjugaison

### Verbes en -ar : hablar

| Pronom | Terminaison | Forme |
|--------|-------------|-------|
| yo | -aba | hablaba |
| tú | -abas | hablabas |
| él / ella / usted | -aba | hablaba |
| nosotros | -ábamos | hablábamos |
| vosotros | -abais | hablabais |
| ellos / ustedes | -aban | hablaban |

### Verbes en -er / -ir : comer / vivir

| Pronom | -er | -ir |
|--------|-----|-----|
| yo | comía | vivía |
| tú | comías | vivías |
| él / ella / usted | comía | vivía |
| nosotros | comíamos | vivíamos |
| vosotros | comíais | vivíais |
| ellos / ustedes | comían | vivían |

## Verbes irréguliers

Seulement trois verbes irréguliers à l'imperfecto :

| Infinitif | yo | tú | él | nosotros | ellos |
|-----------|----|----|----|----------|-------|
| ser | era | eras | era | éramos | eran |
| ir | iba | ibas | iba | íbamos | iban |
| ver | veía | veías | veía | veíamos | veían |

## Quand l'utiliser

| Contexte | Exemple |
|----------|---------|
| Habitude passée | *De niño, comía mucho chocolate.* — Enfant, je mangeais beaucoup de chocolat. |
| Description dans le passé | *La casa era grande y tenía un jardín.* |
| Action en cours interrompue | *Dormía cuando sonó el teléfono.* |
| Heure et âge dans le passé | *Eran las dos. Tenía diez años.* |
| Deux actions simultanées | *Mientras cocinaba, escuchaba música.* |

**Marqueurs temporels typiques :**
siempre · nunca · a veces · todos los días · cuando era niño/a · antes · generalmente · mientras

## Indefinido vs Imperfecto

| Indefinido | Imperfecto |
|------------|------------|
| Action terminée, ponctuelle | Etat, description, habitude |
| *Ayer comí paella.* | *De niño comía paella todos los domingos.* |
| *El lunes fui al médico.* | *Cuando era joven iba al médico con miedo.* |
| Premier plan du récit | Arrière-plan, contexte |

> *Llovía cuando salí de casa.* — Il pleuvait (imperfecto = contexte) quand je suis sorti (indefinido = action).
