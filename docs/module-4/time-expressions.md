# Expressions de temps

## Situer dans le passé

| Espagnol | Français |
|----------|----------|
| ayer | hier |
| anteayer | avant-hier |
| el lunes pasado | lundi dernier |
| la semana pasada | la semaine dernière |
| el mes pasado | le mois dernier |
| el año pasado | l'année dernière |
| hace + durée | il y a + durée |
| hace tres días | il y a trois jours |
| hace un mes | il y a un mois |
| desde hace + durée | depuis + durée |
| entonces | alors, à ce moment-là |
| en aquel entonces | à cette époque-là |
| de repente | soudain |
| de niño/a | enfant, quand j'étais enfant |

## Durée et fréquence

| Espagnol | Français |
|----------|----------|
| durante + durée | pendant + durée |
| durante dos horas | pendant deux heures |
| desde | depuis (point de départ) |
| hasta | jusqu'à |
| desde... hasta... | de... à... |
| siempre | toujours |
| nunca / jamás | jamais |
| a veces | parfois |
| a menudo | souvent |
| raramente | rarement |
| todos los días | tous les jours |
| cada semana | chaque semaine |

## Constructions avec hacer

**Hace + durée + que + présent** — action qui dure encore :
> *Hace dos años que vivo en Francia.* — J'habite en France depuis deux ans.

**Hace + durée + que + indefinido** — action terminée il y a un certain temps :
> *Hace tres días que llegué.* — Je suis arrivé il y a trois jours.

**Llevaba + durée + gerondif** — durée d'une action dans le passé :
> *Llevaba una hora esperando.* — J'attendais depuis une heure.
