# Pronoms et ordre des mots

## Les pronoms sujets

| Espagnol | Français |
|----------|----------|
| yo | je |
| tú | tu |
| él / ella / usted | il / elle / vous (formel) |
| nosotros / nosotras | nous |
| vosotros / vosotras | vous (Espagne) |
| ellos / ellas / ustedes | ils / elles / vous (Amérique latine) |

::: tip
En espagnol, le pronom sujet est souvent omis car la terminaison du verbe suffit à identifier la personne : *hablo* = je parle, *hablas* = tu parles.
:::

::: info usted / ustedes
**Usted** (abrégé **Ud.**) est le vouvoiement formel. Il se conjugue comme la 3e personne du singulier.
En Amérique latine, **ustedes** remplace **vosotros** pour le pluriel.
:::

## L'ordre des mots

L'ordre de base est **Sujet — Verbe — Complément**, comme en français.

> *María come una manzana.* — María mange une pomme.

Mais l'espagnol est plus flexible. On peut inverser pour insister :

> *Una manzana come María.* — C'est une pomme que mange María.

**Les adjectifs se placent généralement après le nom :**

> *un coche rojo* — une voiture rouge
> *una casa grande* — une grande maison

Certains adjectifs changent de sens selon leur position :

| Avant | Après |
|-------|-------|
| *un gran hombre* — un grand homme (remarquable) | *un hombre grande* — un homme grand (physiquement) |
| *mi viejo amigo* — mon vieil ami (de longue date) | *un amigo viejo* — un ami vieux |
