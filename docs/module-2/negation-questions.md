# La négation et les questions

## La négation

En espagnol, on place **no** directement devant le verbe. C'est tout.

> *Hablo español.* — Je parle espagnol.
> *No hablo español.* — Je ne parle pas espagnol.

Pas de "ne...pas" à construire, pas de auxiliaire : juste **no**.

### Double négation

Contrairement au français, la double négation est correcte en espagnol :

| Espagnol | Français |
|----------|----------|
| *No tengo nada.* | Je n'ai rien. |
| *No hay nadie.* | Il n'y a personne. |
| *No voy nunca.* | Je n'y vais jamais. |

Les mots négatifs courants :

| Positif | Négatif | Sens |
|---------|---------|------|
| algo | nada | quelque chose / rien |
| alguien | nadie | quelqu'un / personne |
| siempre | nunca / jamás | toujours / jamais |
| también | tampoco | aussi / non plus |

## Les questions

### L'intonation

La façon la plus simple de poser une question : garder l'ordre des mots et monter l'intonation.

> *¿Hablas español?* — Tu parles espagnol ?

En espagnol écrit, le point d'interrogation ouvrant **¿** est obligatoire.

### Inverser sujet et verbe

On peut aussi inverser :

> *¿Habla usted inglés?* — Parlez-vous anglais ?

### Les mots interrogatifs

| Espagnol | Français |
|----------|----------|
| ¿qué? | quoi / que / quel ? |
| ¿quién? / ¿quiénes? | qui ? |
| ¿dónde? | où ? |
| ¿cuándo? | quand ? |
| ¿cómo? | comment ? |
| ¿cuánto / cuánta? | combien ? |
| ¿cuántos / cuántas? | combien (pluriel) ? |
| ¿por qué? | pourquoi ? |
| ¿para qué? | dans quel but ? |
| ¿cuál? / ¿cuáles? | lequel / lesquels ? |

::: tip
Tous les mots interrogatifs portent un accent écrit pour les distinguer de leurs équivalents conjonctifs : *que* (que), *donde* (où), *como* (comme).
:::

### Exemples

> *¿Dónde está el hotel?* — Où est l'hôtel ?
> *¿Cuánto cuesta?* — Combien ça coûte ?
> *¿Cómo te llamas?* — Comment tu t'appelles ?
> *¿Por qué estudias español?* — Pourquoi tu apprends l'espagnol ?
