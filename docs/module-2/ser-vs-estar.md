# Ser vs Estar

Le français n'a qu'un seul verbe "être". L'espagnol en a deux : **ser** et **estar**.
C'est l'une des difficultés centrales de la langue — mais la règle de base est claire.

## Conjugaison

### Ser

| Pronom | Forme |
|--------|-------|
| yo | soy |
| tú | eres |
| él / ella / usted | es |
| nosotros | somos |
| vosotros | sois |
| ellos / ustedes | son |

### Estar

| Pronom | Forme |
|--------|-------|
| yo | estoy |
| tú | estás |
| él / ella / usted | está |
| nosotros | estamos |
| vosotros | estáis |
| ellos / ustedes | están |

## Quand utiliser lequel

### Ser — identité permanente

| Usage | Exemple |
|-------|---------|
| Identité, nom | *Soy Pierre.* |
| Origine | *Soy de Francia.* |
| Nationalité | *Es española.* |
| Profession | *Es médico.* |
| Caractéristique permanente | *La nieve es blanca.* — La neige est blanche. |
| Heure et date | *Son las tres. Es lunes.* |
| Matière | *La mesa es de madera.* — La table est en bois. |

### Estar — état temporaire ou position

| Usage | Exemple |
|-------|---------|
| Etat physique ou émotionnel | *Estoy cansado.* — Je suis fatigué. |
| Localisation | *Estoy en Madrid.* |
| Résultat d'un changement | *La puerta está abierta.* — La porte est ouverte. |
| Action en cours (+ gerondif) | *Estoy comiendo.* — Je suis en train de manger. |

::: warning Attention
Certains adjectifs changent de sens selon le verbe :

| | Ser | Estar |
|-|-----|-------|
| *aburrido* | il est ennuyeux (de nature) | il s'ennuie (en ce moment) |
| *listo* | il est intelligent | il est prêt |
| *malo* | il est mauvais / méchant | il est malade |
| *bueno* | il est bon (de caractère) | il va bien / il est bon (goût) |
:::
