# Le présent de l'indicatif

## Les trois groupes de verbes

Les verbes espagnols se terminent par **-ar**, **-er** ou **-ir**.

### Verbes en -ar : hablar (parler)

| Pronom | Terminaison | Forme |
|--------|-------------|-------|
| yo | -o | hablo |
| tú | -as | hablas |
| él / ella / usted | -a | habla |
| nosotros | -amos | hablamos |
| vosotros | -áis | habláis |
| ellos / ustedes | -an | hablan |

### Verbes en -er : comer (manger)

| Pronom | Terminaison | Forme |
|--------|-------------|-------|
| yo | -o | como |
| tú | -es | comes |
| él / ella / usted | -e | come |
| nosotros | -emos | comemos |
| vosotros | -éis | coméis |
| ellos / ustedes | -en | comen |

### Verbes en -ir : vivir (vivre)

| Pronom | Terminaison | Forme |
|--------|-------------|-------|
| yo | -o | vivo |
| tú | -es | vives |
| él / ella / usted | -e | vive |
| nosotros | -imos | vivimos |
| vosotros | -ís | vivís |
| ellos / ustedes | -en | viven |

::: tip
Les terminaisons de **-er** et **-ir** sont identiques sauf à **nosotros** et **vosotros**.
:::

## Usages du présent

Le présent espagnol couvre plusieurs usages français :

| Usage | Exemple |
|-------|---------|
| Action en cours | *Trabajo ahora.* — Je travaille maintenant. |
| Habitude | *Como a las dos.* — Je mange à deux heures. |
| Futur proche | *Mañana viajo.* — Demain je voyage. |
| Vérité générale | *El agua hierve a 100°.* — L'eau bout à 100°. |
